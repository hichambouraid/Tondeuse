# Tondeuse
Tondeuse Kata

## Prérequis

- Java 17
- Maven
- IntelliJ IDEA ou eclipse

## Démarrage

Ajouter le mot "input.txt" dans les argument lors du démmarage

## Installation
```bash
# Clonez le dépôt
git clone https://github.com/hichambouraid/Tondeuse.git

# Accédez au répertoire du projet
cd Tondeuse

# Compilez le projet avec Maven
mvn clean install
